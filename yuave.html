Administration Panel

This was an exercise of designing & building a completely new administration panel using VueJS as the JS framework and Zurb Foundation 5 as the HTML framework. The application communicates with a backend server via a token-authorization based JSON api.

The following are screenshots (with private information redacted) of how the Yuave administration panel looks like. A full working demo can be provided upon request.

Customer View

An extensive customer view. This was the key component of this application. The previous version (which was made using Jquery) of the application had an unintuitive customer view, where data was not shown cleary.

I took it upon myself to have the customer data in three distinguishable sections i.e. User Details, Forwarded Numbers, Extensions).
