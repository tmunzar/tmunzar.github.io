Trav2Trav

This was a project taken up by my firm for a travelers-based social network startup called Trav2Trav. We were tasked to build-up the network from the ground up, which included all the branding, UX/UI design, and implementation.

Wireframes

While I was not the visual designer on this project, I was heavily involved in gather user-requirements, determining the scope of the MVP, designing the wireframes (which were tested using Invision), and then finally being the sole engineer to implement the application on Laravel.
