InflightVR

Inflight VR is a Munich based company that is creating a virtual reality ecosystem for the Inflight Entertainment Industry. This young startup wanted our help to make an impression, so we set out to design something that helped display the power & possibities of VR as inflight entertainment.

The design process was a collaboration with a graphic designer. The website was then developed solely by me using CockpitCMS and Foundation 6. Jquery was used to code in the additional effects.

The website is currently live and accessible.
